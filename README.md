nsvlc
=====

nsvlc (Network synchronised VLC) is a playback synchronizer for 2 or more different instances of the VLC media player, running on different machines (Windows or Unix) connected via Internet.

It's purpose is giving friends the ability to watch movies together exactly synchronised, with control events (like start, stop, pause, unpause) automatically synchronised/managed by nsvlc in the background.
